# PropIntelli - Wireframe

Open any of the HTML files directly in your browser:
- `index.html` (Home)
- `onboarding.html` (User Preferences Questionnaire)
- `search.html` (Search Results)
- `property.html` (Property Detail)
- `insights.html` (Neighborhood Insights)
- `analytics.html` (Analytics Tools)
- `auth.html` (Sign-In / Register)
- `dashboard.html` (User Dashboard)
- `compare.html` (Comparison)

Highlights:
- User onboarding questionnaire: area radius, budget range, size slider, purpose selection
- Hierarchical filters (Country → Region → City → Neighborhood)
- Detailed filters: price, size, rooms, type, renovation status
- Insights: crime heatmap/safety score, demographics, infrastructure
- Listings: original links, renovation/resale badges, compare selection + sticky bar
- Analytics: price/sqm, historical trends, ROI estimator
- Dashboard: favorites, compare sets, notifications
- Global floating chat assistant on key pages

All pages share `styles.css` and `app.js`. No build step required.
